# Day 19: 数据复用与缓存优化

## 🎯 学习目标
- 掌握卷积滑窗复用思想
- 了解 On-Chip Buffer 的设计要点

## 一、数据复用策略
- **Weight Reuse**：多个输出共享同一权重
- **Activation Reuse**：滑窗区域共享输入

## 二、卷积滑窗示意
```
输入 Feature Map（3x3 滑窗）

+---+---+---+
| a | b | c |
| d | e | f |
| g | h | i |
+---+---+---+
滑动 1 个单位，下次还可复用 b/c/e/f/h/i
```

## 三、缓存优化原则
- 减少 DRAM 访问
- 提高数据命中率
- 利用片上 SRAM 缓冲多帧数据
