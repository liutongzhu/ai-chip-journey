# Day 20: 带宽瓶颈分析

## 🎯 学习目标
- 理解计算 vs 带宽的瓶颈关系
- 掌握 Roofline 模型分析方法

## 一、Roofline 模型概念
- 横轴：计算密度（FLOPs / Byte）
- 纵轴：性能（FLOPs/s）
- 拐点：内存带宽与峰值计算之间的平衡

## 二、简化 Roofline 图（示意）
```
 ^
 |                    /
 |                   /
 |                  /
 |   带宽瓶颈     /   计算瓶颈
 |---------------/
 +-------------------->
```

## 三、应用分析
- 数据访问频繁，带宽受限
- 算法计算密度低 → 内存优化优先
